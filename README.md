# 🌾 AgriConnect — Smart Farming Assistant

AgriConnect is an AI-powered web platform built to support farmers with crop disease detection, agricultural knowledge, community interaction, and more. Designed with scalability, multilingual support, and modern UI/UX, it's crafted to bring technology closer to agriculture.

---

## 🚀 Features

- 🌿 **Crop Disease Detection** (via AI image recognition)
- 🧠 **Knowledge Base** for curated articles and tips
- 💬 **Chatbot Assistant** for real-time farming queries
- 🌍 **Community Forum** for farmer interaction
- 🗣️ **Multilingual Support** for regional accessibility
- 🛍️ **Product Recommendations** with potential monetization
- 📶 **Offline-ready** via PWA support

---

## 🛠️ Tech Stack

| Layer       | Technology                        |
|-------------|-----------------------------------|
| Frontend    | React.js, Tailwind CSS, ShadCN UI |
| Backend     | Node.js, Express.js               |
| Database    | MongoDB Atlas                     |
| AI          | Custom-trained ML models          |
| Deployment  | Vercel, Docker-ready              |
| Dev Tools   | PNPM, TypeScript                  |
